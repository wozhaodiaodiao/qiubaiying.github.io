    使用方法

    首先介绍整体开发思路吧

    首先项目开始请求一个活动信息的时候，后端就已经把分享链接传过来了，前端要做就是全局存储这个值，然后通过微信分享的时候讲这个链接配置给分享卡片发出去，然后其他用户通过该卡片所提供的链接进入，不过有个大坑，接下来会提供解决方法

    1.导入：

    • cdn方式，项目根目录下index.html引入
    • npm

    2.开发wxapi组件

    通过分享卡片进来之后有个坑，请注意

    首先上面提到的思路，用户通过分享卡片进来之后，链接居然被微信给改动了，不能够正确跳转到裂变海报工具的地址，还需要对这个链接进行一些字符串操作，才可以

    首先你配置好的链接，微信会拼接的字符串是
    ?from=singlemessage或者?from=singlemessage&isappinstalled=0
    你应该做一定处理，代码如下：

    //微信分享默认会带?from=singlemessage，需要替换掉然后跳转过去
    let url = document.location.toString();
    if (url.indexOf("?from=singlemessage") != -1) {
      var a = url.replace("?from=singlemessage", "");
      a = a.replace("&isappinstalled=0", "");
      window.location.href = a;
      return;
    }
